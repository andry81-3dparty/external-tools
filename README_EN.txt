1. DESCRIPTION
-------------------------------------------------------------------------------
External command line tools collected into a single project but with different
repositories. A standalone repository is allocated per each tool but without
a default branch. Each branch does include at least a tool major version.

The latest version is here: https://sf.net/p/external-tools

WARNING:
  Use the SVN access to find out latest functionality and bug fixes.
  See the REPOSITORIES section.
